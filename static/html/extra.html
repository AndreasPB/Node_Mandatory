Heroku

Jeg valgte Heroku fordi det er en gratis løsning der virker med Node.

Opsætning:

Heroku leder et et script i package.json der hedder "start", tilføjede "start": "node app.js."

For at håndterer porten, nu hvor vi ikke kører på localhost, tilføjer jeg "process.env.PORT" til min port, som min app.listen bruger.

const port = process.env.PORT || 8080; 

app.listen(port, (error) => {
    if (error) {
        console.log("Error starting the server");
    }
    console.log("Server is running on port", port);
});

Man kan bruge deres CLI(Command Line Interface), men brugte deres side til at forbinde til mit GitHub.
